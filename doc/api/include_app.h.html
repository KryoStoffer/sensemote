Application Entry Point

Callbacks made from OS to app code. App should avoid blocking wherever possible, by scheduling its own tasks.

app_init

   void app_init(void)

Initialise application. Will only be called once, on startup.

app_tick

   void app_tick(void)

Callback to indicate a system tick.

app_100hz

   void app_100hz(void)

Callback to indicate passing of 0.01s.

app_10hz

   void app_10hz(void)

Callback to indicate passing of 0.1s.

app_1hz

   void app_1hz(void)

Callback to indicate passing of 1s.
