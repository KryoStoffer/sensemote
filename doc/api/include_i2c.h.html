I2C

Support for I2C master.

i2c_init

   void i2c_init(void)

Initialise i2c pins.

i2cwriteregister

   uint8_t i2c_write_register(uint8_t devaddr, uint8_t regaddr, uint8_t d)

Write 8 bit value to register of i2c device.

Parameters

  • devaddr [in]
    • Device I2C address
  • regaddr [in]
    • Register address
  • d [in]
    • Data to write

Returns

  • 1 for success, 0 for error
