ADC

Support for Analogue to Digial Converter

adc_init

   void adc_init(void)

Initialise ADC hardware.

adc_read

   uint16_t adc_read(uint8_t channel, BOOLEAN externalRef);

Poll a single ADC channel and perform a single conversion (~132μS).

Parameters

  • channel [in]
    • ADC channel to read
  • externalRef [in]
    • Use external voltage reference

Returns

  • 12bit ADC reading

adc_readBattery

   uint16_t adc_readBattery(void)

Poll battery voltage.

Returns

  • Battery voltage in mV.
