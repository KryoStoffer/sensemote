Simulator

Everything in the SenseMote firmware tree builds for both CC1110 and as a simulator to run on a PC. Without any XRF hardware, the core OS and apps can be tested. The radio is simulated over IP multicast and the serial port is brought out to the console.

To build an app for the PC and run it, type make APP=name PLATFORM=POSIX install

To test the Pachube features. Configure config.mk with a Pachube feedid and API key, then open two windows. In one do make APP=pbapp PLATFORM=POSIX install. In another do make APP=pbnode PLATFORM=POSIX install. Type help for information on the commands available. Next, edit config.mk and change the EUI64 (the radio address of the node) and run a second copy of pbnode again with make APP=pbnode PLATFORM=POSIX install. Experiment with subscribing to keys in one app and changing them in the other.
