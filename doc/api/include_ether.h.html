Ethernet

Support for ethernet.

ether_init

   void ether_init(void)

Initialise ethernet device.

ether_tick

   void ether_init(void)

Advance state of ethernet, should be called regularly.

ether_tx

   void ether_tx(uint8_t *buf, uint16_t len);

Transmit an ethernet frame.

Parameters

  • buf [in]
    • Ethernet frame
  • len [in]
    • Frame length

ether_rx

   uint16_t ether_rx(void)

Attempt to receive an ethernet frame. Should be called regularly, to prevent overflow. Frame data will be written directly to uIP's input buffer.

Returns

  • Length of received frame

ether_connected

   void ether_connected(BOOLEAN connected)

Callback made to user to indicate ethernet link state change.

Parameters

  • connected [in]
    • TRUE
      • Link up
    • FALSE
      • Link down
