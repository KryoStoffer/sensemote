SenseMote

SenseMote is an operating system and set of applications for the ChipCon CC1110 SoC radio. Reference designs for a low-cost Internet gateway and sensors nodes are provided.

Hardware

SenseMote is designed for the ChipCon CC1110. The CC1110 is a single chip device featuring a flexible sub-1Ghz radio and an 8051 microcontroller.

Currently, all SenseMote reference designs are based on the Ciseco XRF CC1110 module.
